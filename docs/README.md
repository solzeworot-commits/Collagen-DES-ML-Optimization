# Диссертация Одиловой З.А. — Полностью воспроизводимый проект

## Структура проекта
- `data/` — исходные и обработанные данные
- `scripts/` — все вычислительные скрипты
- `notebooks/` — Jupyter-ноутбуки для анализа
- `models/` — сохранённые модели (.pkl)
- `results/` — все таблицы и графики из приложений
- `docs/` — документация
- `tests/` — тесты воспроизводимости
